# phalcon-docker
PHP Debian bookworm image with Phalcon installed 

Installed extensions
- gd
- xdebug
- imagick
- gettext
- pdo_mysql
- sodium
- redis
- amqp
- decimal
- apcu
- zephir_parser
- phalcon
